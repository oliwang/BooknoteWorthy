Export Apple Books Annotations with Style


Step 1: Upload Database Files

Upload your Apple Books Database Files

Location

.sqlite files in /Users/<UserName>/Library/Containers/com.apple.iBooksX/Data/Documents/BKLibrary and /Users/<UserName>/Library/Containers/com.apple.iBooksX/Data/Documents/AEAnnotation

Hidden Library Folder

On Macs you might need to Unhide the Library Folder before uploading.

Create Aliases

You can create aliases of these two database files for easier uploading in the future.

Privacy

Nothing is uploaded to the server.


Step 2: Set Template

Set the template for your exported notes

Use the following variables in your template:


{{book.title}} - Title of the book

{{book.author}} - Author of the book


{{#each annotations}}{{/each}} - Loop through annotations

{{assetid}} - The book's assetid

{{annotation_location}} - The location in the book

{{selected_text}} - Highlight

{{note}} - Note you took

{{ibook_link}} - deeplink to selected text
